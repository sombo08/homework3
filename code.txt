null + 3

Результат: 3
Коментар: Число null перетворюється в 0, тому отримуємо 0 + 3 = 3

5 && "qwerty"

Результат: "qwerty"
Коментар: Оператор && повертає останнє значення, у цьому випадку "qwerty".

+'40' + +'2' + "hillel";

Результат: "42hillel"
Коментар: Унарний оператор + перетворює рядки '40' і '2' в числа 40 і 2, після чого вони додаються разом, а потім додається рядок "hillel".

'10' - 5 === 6;

Результат: false
Коментар: Рядок '10' перетворюється в число 10, від якого віднімається 5, що дає 5, але це не дорівнює 6, тому отримуємо false.

true + false

Результат: 1
коментар: Булеві значення перетворюються в числа (true = 1, false = 0), тому true + false дає 1.

'4px' - 3

'4' - 3

Результат: 1
Коментар: Рядок '4' перетворюється в число 4, від якого віднімається 3, що дає 1.

'6' + 3 ** 0

Результат: "61"
Коментар: Рядок '6' конкатенується з 3 ** 0 (що дорівнює 1).

12 / '6'

Результат: 2
Коментар: Рядок '6' перетворюється в число 6, тому 12 / 6 дає 2.

'10' + (5 === 6);

Результат: "10false"
Коментар: Результат порівняння (5 === 6) дорівнює false, що конкатенується з рядком '10', отримуючи "10false".

null == ''

Коментар: Рівність null == '' не виконується

3 ** (9 / 3);

Результат: 27
Коментар: 9 / 3 дорівнює 3, тому 3 ** 3 дає 27.

!!'false' == !!'true'

0 || '0' && 1

Результат: 1
Коментар: Вираз 0 || '0' && 1 оцінюється як false || true && true, що дає true.

(+null == false) < 1;

Результат: false
Коментар: Вираз (+null == false) оцінюється як true, а потім порівнюється з числом 1, що дає false, оскільки 1 не менше 1.

false && true || true

Результат: true
Коментар: Вираз false && true || true це false || true, що дає true.

false && (false || true);

Результат: false
Коментар: Вираз false || true дорівнює true, але false && true дорівнює false, отже, отримуємо false.
